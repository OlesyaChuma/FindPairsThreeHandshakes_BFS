# Поиск пар через три рукопожатия (BFS)

## Описание проекта

Данный проект моделирует простую социальную сеть с использованием теории графов на языке C++.

Каждый человек представляется в виде **вершины графа**, а каждое знакомство — в виде **неориентированного ребра**.  
Основная цель проекта — найти все пары людей, которые знакомы **не более чем через три рукопожатия**, что соответствует кратчайшему расстоянию между вершинами графа, не превышающему 3.

Проект выполнен в **учебных целях** и основан на базовых темах курса алгоритмов и структур данных.

---

## Используемые понятия

- Графы
- Неориентированный граф
- Список смежности
- Поиск в ширину (BFS)
- Кратчайший путь в невзвешенном графе

---

## Детали реализации

- **Вершины графа** — люди
- **Рёбра графа** — знакомства между людьми
- Граф реализован с помощью **списка смежности**
- Для поиска кратчайших расстояний используется **поиск в ширину (BFS)**
- Глубина поиска ограничена **тремя уровнями** согласно условию задачи

---

## Сборка и запуск

Проект предназначен для компиляции в **Visual Studio**.

1. Создайте проект типа **Console Application**.
2. Добавьте в проект файлы `Graph.h`, `Graph.cpp` и `main.cpp`.
3. Соберите и запустите программу.
